double get totalPrecio

Calcula el precio total de la orden multiplicando precio por cantidad de cada producto y sumando.

Implementation

double get totalPrecio {
  if (productos.isEmpty) {
    return 0.0;
  } else {
    return productos
        .map((producto) => producto.precio * producto.cantidad)
        .reduce((a, b) => a + b);
  }
}
